FastAPI-Azure-Auth provides support for both v1 and v2 tokens, but we'll only document and have a tutorial
for v2 tokens.

To read about the difference between `v1` and `v2` tokens, check out
[this article](https://docs.microsoft.com/en-us/azure/active-directory/develop/access-tokens).

If you already have your application setup in Azure with v1 tokens and want to migrate, you can do so by following
the **link here** steps.
